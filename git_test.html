git init -> מתקין גיט בפרוייקט

Git add . -> מוסיף את כל הקבצים הנוכחים לבראנץ המאסטר בברירת מחדל

Git commit -m "message" -> סוגר בחבילה ומאשר לענף להוסיף את כל הקבצים שהוספנו באד

Git branch -> מציג את כל הענפים/בראנצים שיש במערכת

Git branch branch_name ->מייצר ענף חדש בשם שניתן לו בבראנצ'' ניים

git checkout branch_name -> יעבור לבראנץ המבוקש

אם יש שינוי ונרצה לעבור לבראנץ אחר או לייצר חדש
