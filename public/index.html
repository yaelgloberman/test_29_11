Study - Routes

Some "GET" requests you can get information about the API, By adding these queries you can see the edit

  • Per-Page : you can select the amount of obj per one page.
  • Page : you can select the num of pages you want.
  • Sort : you can select how to make your sort - by Price / ID / Name / Cat.
  • Reverse : you can choose the sort dirction.

Domain https://api-toys.cyclic.app

#Method URL Body Description
GET /study none Get a list of all the study, will present 10 results per page.
GET /studyRequests/search?topic=bible&language=english none search for a study request by searching follwing name after the "s=", it will present 10 results per page.
GET /studyRequests/duration?min=12&max=25 none Get a list of the study request by min & max hourse , it will present 10 results per page.
GET /studyRequests/topic/:topicName none Get a list of all study requests according to the selected topic category, it will present 10 results per page.
